# Proyecto 2 – Resumen ejecutivo de ventas de Walmart

## 🎯 Objetivo
Analizar los datos de ventas de Walmart y generar un resumen ejecutivo que muestre los principales indicadores de negocio.

## 📂 Dataset
- Fuente: [indica si fue proporcionado por el bootcamp, Kaggle, etc.]
- Tamaño: [ejemplo: 50,000 registros]
- Variables principales: fecha, producto, categoría, cantidad, precio, región.

## 🔧 Herramientas utilizadas
- SQL para consultas y agregaciones
- Excel / Power BI para visualización
- Tablas dinámicas y gráficos ejecutivos

## 📊 Consultas / pasos principales
1. Ventas totales por categoría de producto.
2. Ingresos por región y periodo.
3. Top 10 productos más vendidos.
4. Creación de dashboard ejecutivo.

## 📈 Resultados
- Identificación de las categorías con mayor participación en ventas.
- Regiones con mayor crecimiento en ingresos.
- Productos clave que generan la mayor parte de las ventas.

## 💡 Aprendizajes
- Cómo transformar datos brutos en indicadores ejecutivos.
- Uso de SQL combinado con herramientas de visualización.
- Importancia de presentar resultados claros para la toma de decisiones.
